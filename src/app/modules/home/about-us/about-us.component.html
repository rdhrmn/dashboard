About


Angular 2 App with a Responsive Bootstrap Nav Bar including Page Routing (3.3.1)

  • Angular 2 (v2.3.1)
  • Angular Router (v3.3.1)
  • TypeScript (v2.1.10)
  • Bootstrap 4 (v4.0.0-alpha.5)
  • BootStrap CSS
  • ng-Bootstrap
  • SASS (v.1.1)

This a developed using Component Driven Design (CCD) and Angular/ES6 Modules. The main modules makes use of custom modules that provide the Navigation Bar and Page/Routing Modules. Also used is Barrels within Angular to help reduce Import Noise.
